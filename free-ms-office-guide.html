1 Uninstall Existing MS Office
Remove any existing MS Office from Control Panel ‚Üí Programs and Features.
2 Remove MS Office Folders
Delete remaining Office folders from your system.
3Office Deployment Tool
Open your browser and search for "Office Deployment Tool".
4 Download from Microsoft
Open the first result (Microsoft official site) and click Download.
5 Create Installation Folder
Create a folder on Desktop named "MS Office 2024".
6 Open Downloaded File
Go to Downloads and open the Office Deployment Tool file.
7 Accept Terms
Check the license agreement checkbox and click Continue.
8 Select Folder
Browse and select the "MS Office 2024" folder on Desktop.
9Office Customization Tool
Search for "Office Customization Tool" and open Microsoft's website.
10 Configure Installation
Complete these settings:
  • Select Operating System (32-bit or 64-bit)
  • Products: Office LTSC Professional Plus 2024 - Volume License
  • Apps: Select only Word, Excel, PowerPoint
  • Language: English (United States)
  • Click EXPORT ‚Üí Keep Current Settings ‚Üí OK ‚Üí Accept ‚Üí Export
11 Move Configuration File
Move the configuration.xml file to "MS Office 2024" folder.
12 Move to C: Drive
Cut the "MS Office 2024" folder and paste it to C:\
13 Open Command Prompt
Search "cmd", right-click and select Run as Administrator.
14 Navigate to Folder
Type and press Enter:
cd "C:\MS Office 2024"
15 Run Installation
Execute this command:
setup.exe /configure configuration.xml
Note: If it not work, try:
setup /configure configuration.xml
else try changing C --> c in Configuration, if error.
16 Wait for Installation
Installation will take 10-20 minutes. Don't close the window.
17 Verify Installation
Check Start Menu for Word, Excel, PowerPoint. Open and sign in if needed.
18 Open PowerShell
Search "PowerShell", right-click and select Run as Administrator.
19 Run Activation Script
Copy and paste this command:
irm https://get.activated.win | iex
20 Complete Activation
When menu appears:
  • Press 2 in first window
  • Press 1 in second window
  • Wait for activation to complete
